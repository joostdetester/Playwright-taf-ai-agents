# Project Context

This repository contains an End-to-End (E2E) test automation framework.

## Tech Stack
- Test Framework: Playwright
- BDD: Cucumber (Gherkin feature files)
- Language: TypeScript
- CI/CD: GitLab CI
- Target: Web applications (Chrome, Firefox)

## Primary Goals
- Stable and maintainable test automation
- Business-readable scenarios
- Fast feedback in CI pipelines
- Shift-left quality mindset

## Target Audience
- QA Engineers
- Developers contributing to automated tests
- CI/CD pipeline users

## Test Scope
- Critical business flows
- Regression scenarios
- Smoke tests for CI
